The sandwich method!
This isn't new. Some people call this the belt method.
I don't use this method, but here goes!

Layer separation (0)
  1. You should plan a 2x2x3 + 2 E-edges in inspection
  2. Then do the rest of separation with just R, U, and F moves (for EO)
Canonical Sandwich (1)
  1. Separate the layers - D edges in D, E edges solved in E, and U edges in U
  2. OLL + PLL on D - Note that there can be parity cases that are not normal OLLs / PLLs, increasing the number of OLLs to 347 and the number of PLLs to 43. OLL would almost certainly be done in two looks.
  3. OLL + PLL on U - Normal OLL (57 algs) / PLL (21 algs)
Concurrent Sandwich (2)
  1. Separate the layers - Same as in Canonical
  2. Orient edges on both layers
  3. Orient corners on both layers
  4. Position corners on both layers
  5. Position edges on both layers
EO Sandwich (3)
  1. EO, zz-style
  2. Separate the layers without breaking EO
  3. Continue in the same fashion as Concurrent
Belt Sandwich (4)
  1. EO-Belt
  2. Separate the layers using only U, D, R2, L2, F2, and B2 moves
  3. Continue in the same fashion as Concurrent
3/4 Sandwich (5)
  1. EO + 3/4 belt
  2. Separate the layers minus one corner without breaking the 3/4-belt
  3. Insert the last corner with the last E-edge with either R' U R, R' U2 R, or r' U r' F' r2 to avoid "parity"
  4. COLL on both layers
  5. Position edges on both layers
